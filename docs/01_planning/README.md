# 01 Planning (기획 및 비즈니스 도메인)

> 서비스 확장성, 신규 테스트 기획 등 비즈니스 로드맵 명세서 보관소

본 디렉터리 하위의 문서들은 에이전트 및 팀원들이 참조할 수 있도록 항상 최신의 Frontmatter(수정 날짜 등)를 유지해야 합니다. 
신규 문서를 작성할 때에는 아래 **[표준 작성 템플릿]**을 복사하여 사용하십시오.

---

## 📄 신규 기획서 작성 템플릿 (Template)

해당 템플릿은 `tests/` 하위의 신규 심리테스트 기획이나 `foundation/` 하위의 핵심 확장성 기획 시 활용됩니다. 코드 블록 내용을 복사하여 새 `.md` 파일에 붙여넣고 항목들을 채워주세요.

```markdown
---
title: "[기획서 제목]"
author: "[작성자명 또는 직책]"
created_date: "YYYY-MM-DD"
last_updated: "YYYY-MM-DD"
status: "Draft | Review | Active"
dependencies: ["참조할 기존 기획서 slug (선택)"]
---
```
