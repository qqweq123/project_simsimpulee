# Demon Slayer (Taisho Modern) Design Concept

## 1. Keywords
- **Taisho Roman (다이쇼 로망)**: 서양 문물과 일본 전통이 섞인 개화기 감성.
- **Sumi-e (수묵화)**: 강렬하고 거친 먹물의 흔적.
- **Ethereal (몽환적)**: 등꽃이 흩날리는 신비로운 분위기.

## 2. Color Palette
| Role | Color | Hex | Description |
|---|---|---|---|
| **Background** | **Midnight Ink** | `#0b0f19` | 깊은 밤, 먹물 같은 어둠 |
| **Surface** | **Rice Paper** | `#f5f2e8` | 오래된 종이 질감 (텍스트 배경) |
| **Primary** | **Blade Blue** | `#3b82f6` | 물의 호흡, 차가운 이성 |
| **Secondary** | **Demon Red** | `#ef4444` | 혈귀, 불의 호흡, 강렬함 |
| **Accent** | **Wisteria** | `#a78bfa` | 등꽃, 신비, 고귀함 |
| **Text** | **Sumi Black** | `#1c1917` | 수묵의 진한 검정 |

## 3. Visual Elements
- **Typography**:
    - Title: `Noto Serif KR` (Bold, 800) - 힘있는 명조체
    - Body: `Noto Sans KR` (Light/Regular) - 현대적인 고딕체
- **UI Components**:
    - **Buttons**: 마름모 꼴(Diamond shape) 또는 붓터치 배경.
    - **Cards**: 한지(Hanji) 텍스처와 금박 테두리.
    - **Inputs**: 세로쓰기(Vertical writing) 느낌을 주는 레이아웃 배치.

## 4. Interaction (Breath)
- **Loading**: 칼을 뽑는 소리(Visual) 또는 호흡 집중 효과.
- **Selection**: "전집중 호흡" 처럼 화면이 순간적으로 집중되었다가 베어지는 효과 (Slash).
- **Background**: 은은하게 피어오르는 연기(Smoke) 또는 반딧불이.

## 5. Result Page Structure
- **Character Reveal**: 연기가 걷히며 캐릭터가 등장.
- **Background**: 해당 캐릭터의 속성(물, 불, 번개 등)에 맞는 배경 이펙트.
- **Stats**: 육각형 방사형 차트(Radar Chart)로 능력치 표현.
