# Neo-Brutalism Style Guide

## Concept
- **Keywords**: Pop, Bold, Raw, Trendy
- **Mood**: 격식 없고 솔직한, 에너지가 넘치는, 디지털 네이티브 감성.

## Visual Language
- **Layout**:
    - Hard grids, overlapping elements
    - "Raw" HTML aesthetics (default inputs, courier fonts) mixed with modern colors
- **Shapes**:
    - Geo-shapes (circles, stars, rectangles)
    - **Thick black borders (3px~5px)** on everything
    - Sharp drop shadows (no blur, offset: 4px 4px)
- **Typography**:
    - Display: `Black Han Sans` or ultra-bold sans-serif
    - Body: `Courier New` or `Roboto Mono` (Monospaced)
- **Colors**:
    - High contrast: High saturation colors against distinct backgrounds
    - Palette: Neon Green (#CCFF00), Hot Pink (#FF00CC), Cyan (#00FFFF)
    - Text: Pure Black (#000000)

## UI Elements
- **Buttons**: Looks like physical blocks, click interaction moves the shadow
- **Badges**: Marquee text (scrolling text), sticker-like visuals
- **Images**: Halftone patterns, cut-out collage style
