# Minimal Editorial Style Guide

## Concept
- **Keywords**: Sophisticated, Clean, Timeless, Professional
- **Mood**: 고급 라이프스타일 매거진, 갤러리 도록, 본질에 집중하는.

## Visual Language
- **Layout**:
    - **Whitespace**: Extreme use of padding and margin
    - Grid-based alignment (Swiss Style)
    - Asymmetric balance
- **Typography**:
    - Contrast between large Display fonts and small, functional body text
    - Mix of Serif and Sans-serif
- **Imagery**:
    - High-quality B&W photography
    - Or no images, just typography
- **Colors**:
    - Monochrome: Black (#000000), White (#FFFFFF), Off-white (#FAFAFA)
    - Single Accent: International Orange (#FF4F00) or Royal Blue (#4169E1)

## UI Elements
- **Dividers**: Fine 1px lines separating sections
- **Typography**: Big, bold numbers (e.g., "01", "Type A")
- **Interaction**: Smooth, inertia-based scrolling feel.
