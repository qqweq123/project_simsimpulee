# Cyberpunk Tech (HUD) Style Guide

## Concept
- **Keywords**: Futuristic, Data-driven, Glitch, Neon
- **Mood**: SF 영화의 시스템 인터페이스, 분석적, 차가운 지성.

## Visual Language
- **Layout**:
    - Technical lines, crosshairs, grid overlays
    - HUD (Heads-Up Display) elements
    - Corner brackets on containers `[ Content ]`
- **Typography**:
    - Monospaced fonts (`Share Tech Mono`, `Fira Code`)
    - All Caps for headers
    - Scrambling text effect on load
- **Colors**:
    - Background: Deep Black (#000000) or Dark Gray (#111111)
    - Primary: Neon Cyan (#00FFFF), Matrix Green (#00FF00)
    - Alert: Neon Red (#FF0033)
- **Effects**:
    - **Glitch**: Random chromatic aberration (RGB split) interactions
    - **Scanlines**: Horizontal lines moving down screen (CRT monitor effect)
    - **Blink**: Elements blinking like status indicators

## UI Elements
- **Progress Bars**: segmented blocks
- **Charts**: Radar charts (hexagon shape)
- **Buttons**: Hollow with glowing borders
